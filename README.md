# Taller Aleilla — Aelia Motor Web

Next.js 14 · Tailwind CSS · TypeScript

## Setup
```bash
npm install
npm run dev
```

## Deploy
Conecta este repo a Vercel. Se despliega automáticamente con cada push a `main`.

## Fotos
Coloca las imágenes en `public/images/`:
- `taller-fachada.jpg` — foto exterior (Hero background)
- `taller-interior.jpg` — interior del taller
- `taller-hero.jpg` — vista general
- `taller-recepcion.jpg` — recepción
